# pyzhon.zsh-theme

[![asciicast](https://asciinema.org/a/45917.png)](https://asciinema.org/a/45917)

## install
```zsh
cd ~/.oh-my-zsh/themes
wget https://raw.githubusercontent.com/orggue/.dotfiles/master/zsh/pyzhon.zsh-theme
```
change in your *.zshrc* your theme to:
```zsh
ZSH_THEME="pyzhon"
```
